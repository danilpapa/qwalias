# qwalias

Простой TUI-инструмент на Rust (`ratatui`) для добавления alias в `~/.zshrc`.

## Запуск

```bash
cargo run
```

## Как пользоваться

1. В поле `name` введите имя alias (например: `root`).
2. В поле `call` введите команду (например: `cd`).
3. Нажмите `Enter` для установки.

## Управление

- `Tab` — переключить поле
- `Backspace` — удалить символ
- `Enter` — запустить установку alias
- `q` / `Esc` / `Ctrl+C` — выход

## Скрипт

Используется `install_aliases.sh`:

- добавляет или обновляет alias в `~/.zshrc`
